Actividades:

1: La etiqueta principal utilizada para crear una tabla en "html", seria la etiqueta "table".

2: La etiqueta que se utiliza para definir una fila, seria la etiqueta "tr".

3: La etiqueta que se utiliza a la hora de crear una celda de encabezado en una tabla, seria la etiqueta "th".

4: La etiqueta utilizada para crear una celda de datos en una tabla, seria la etiqueta "td".

5: El atributo que se utiliza para especificar la cantidad de columnas que deberia tener una celda, seria el atributo "colgroup".

6: El atributo que se deberia de utilizar para especificar la cantidad de celdas, es el atributo "colspan".

7: Para crear una tabla de 10 filas, se deberia utilizar la eiqueta "tr" y se deberia utilizar 10 veces.
